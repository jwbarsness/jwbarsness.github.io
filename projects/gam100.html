StarScavengers

A space exploration game where the player has to travel to asteroids, mine them, and return to their base while fending off enemies.

Role: Game Artist/UI Designer
Tech: C++
Focus: Sprite Art, Menus

My Contributions

  • Made various sprites for player, enemies, asteroids, and materials.
  • Managed the start menu, game menu, HUD menu, and other UI

Gameplay Systems

  • Projectile System
  • Risk-Reward Combat
  • Upgrades
  • Collision System

Challenges & Solutions

The collision/movement of the player required precise calculations from everyone. I worked on making sure sprites matched their game state to keep the immersion
